# ScanMoney
PayPal Hack-ATL 2018
* 3rd Place
* Most Creative
* Biggest Wow Factor

Using AWS Rekognition and ARCore, simply wave your phone in front of a pile of bills to automatically count them and display their values on top of each bill.
